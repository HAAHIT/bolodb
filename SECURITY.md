## Reporting a Vulnerability

BoloDB handles database credentials and query data. If you discover a security vulnerability, please report it privately.

**Do not open a public issue.** Instead, email the maintainers or reach out via the project's discussion channel.

We will acknowledge receipt within 48 hours and work on a fix. You will be credited for the discovery unless you wish to remain anonymous.
