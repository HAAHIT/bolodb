01 — House rules

Six rules the whole page obeys

A

Transform & opacity only

Never animate layout or paint properties on scroll. GPU-composited moves stay at 60fps.

B

One easing family

Entrances use --ease. Playful pops use --spring. Nothing else.

C

Stagger, don't dump

Reveal groups with 40–80ms offsets. Cap any sequence near 600ms so nothing feels slow.

D

Enter once

Content reveals fire a single time. Only ambient and looping elements repeat.

E

In-view only

Anything that loops pauses when off-screen. A battery requirement, not a nicety.

F

Correct with zero JS

Final states are the CSS default. Reduced-motion and no-JS both land on a finished page.

03 — Micro-interaction lab

The tactile vocabulary

Each ships as a reusable Svelte use: action. All degrade to nothing on touch and under reduced-motion.

Magnetic button

The label drifts toward the cursor within ~80px, then springs home. gsap.quickTo, max ~12px.

High confidence

3D tilt + spotlight

Cards read the pointer: ≤6° rotate, a radial highlight, a lit gradient border. This whole card is doing it right now.

0

Number count-up

Verified answers tick up when they scroll into view. tabular-nums so digits never jitter.

Talk to your database like a human.

Masked text reveal

Headlines rise line-by-line behind an overflow mask (SplitText). The LCP line never fades from zero.

Underline wipe

Links grow an underline from the left on hover. scaleX 0→1, brand color, cheap and everywhere.

Scroll-draw path

The pipeline spine and flywheel draw themselves via strokeDashoffset, scrubbed to scroll.
